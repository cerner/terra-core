# Anchor Button

The subcomponent of [Terra-Button](https://engineering.cerner.com/terra-ui/components/cerner-terra-core-docs/button/about) that can display a button rendered using anchor element.

The `action` variant is intended for specific solutions in which a non-inline floating button sits above the page content in a fixed position.

## Getting Started

- Install with [npmjs](https://www.npmjs.com):
- `npm install terra-button`
