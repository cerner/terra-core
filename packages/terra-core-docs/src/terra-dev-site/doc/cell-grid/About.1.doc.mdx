# Terra Cell Grid

<Notice variant="deprecation">

`terra-cell-grid` is no longer supported. If you are using `terra-cell-grid`, consider replacing it with a supported component.

The terra-cell-grid is a structural row based layout for aligning columns with 100% of its parents width.

## Getting Started

- Install with [npmjs](https://www.npmjs.com):
  - `npm install terra-cell-grid`
