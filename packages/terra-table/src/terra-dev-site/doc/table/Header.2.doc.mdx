# Terra Table - Header

The table's header is comprised of a cells and an optional selectAllColumn. The checkLabel should be provided if the table has toggle behavior.

## Getting Started

## Usage

```jsx
import React from 'react';
import Table from 'terra-table';

<Table
  headerData={{
    selectAllColumn: {
      checkStatus: 'empty',
      checkAlignment: '2rem',
      checkLabel: 'Batch Label Example',
    },
    cells: [
      { id: 'header-name', key: 'name', children: 'Name' },
    ],
  }}
/>
```
