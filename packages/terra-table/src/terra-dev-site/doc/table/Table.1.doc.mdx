# Terra Table

Terra Table is a structural component used to create data tables. Table provides means to handle row selection and hooks for sortable columns.

## Getting Started

- Install with [npmjs](https://www.npmjs.com):
  - `npm install terra-table`

## Usage

```jsx
import React from 'react';
import Table from 'terra-table';

<Table
  summaryId="example-id"
  summary="Example table summary."
  headerData={{
    cells: [
      { id: 'header-name', key: 'name', children: 'Name' },
    ],
  }}
  sectionData={[
    {
      rows: [
        {
          key: 'row-0',
          cells: [
            { key: 'cell-0', children: 'John Smith' },
            { key: 'cell-1', children: '123 Adams Drive' },
            { key: 'cell-2', children: '111-222-3333' },
          ],
        },
      ],
    }
  ]}
/>
```
