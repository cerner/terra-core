# Terra Form Tag Select Field

A convenience wrapper assembling a [terra-form-select TagSelect](https://github.com/cerner/terra-core/tree/main/packages/terra-form-select/src/TagSelect.jsx) within a [terra-form-field](https://github.com/cerner/terra-core/tree/main/packages/terra-form-field).

## Getting Started

- Install with [npmjs](https://www.npmjs.com):
  - `npm install terra-form-select`

## Usage

```jsx
import TagSelectField from 'terra-form-select/lib/TagSelectField';
```
