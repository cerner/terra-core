# Terra Text

Component for displaying customizable text in UI.

## Getting Started

- Install with [npmjs](https://www.npmjs.com):
  - `npm install terra-text`

## Usage

```jsx
import React from 'react';
import Text from 'terra-text';

<Text>Hello World</Text>
```
